# Covid Dashboard



## Developers

If you want to run the latest code from git, here's how to get started:

1. Clone the code:

        git clone https://github.com/renuka8/covid-dashboard.git
        cd covid-dashboard

2. Install the node-red dependencies

        npm install


3. Run

        ng serve
